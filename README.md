Минималистичный и стильный системный фечер для macOS, созданный как современная и упрощенная альтернатива neofetch. Скрипт использует нативный протокол графики терминала **Kitty** для рендеринга изображений (включая анимированные GIF) и динамически подстраивает текстовые блоки без использования ASCII-арта.

##  Особенности

* ** Нативная графика Kitty:** Прямой вывод изображений и GIF-анимации высокого качества.
* ** Режим анимации:** Поддержка живых обоев через флаг `--gif`.
* ** Приветственный режим:** Персонализированный вывод с датой, временем и умным расчетом времени работы батареи (`--hello`).
* ** Deep Clear:** Полная очистка экрана и буфера прокрутки перед каждым запуском.
* ** Умное выравнивание:** Автоматический расчет отступов, чтобы текст никогда не наезжал на изображение.
* ** Mac-specific:** Глубокая интеграция с macOS (модель MacBook, статус GPU, здоровье батареи).

## Быстрая установка

Самый простой способ — выполнить одну команду в терминале. Скрипт автоматически скачает файлы, настроит алиас и проверит зависимости.

```bash
curl -fsSL https://raw.githubusercontent.com/alezhg1/apathyfetch/master/install.sh | bash
```

После установки перезапустите терминал или примените изменения командой:
`source ~/.zshrc`

> **Важно:** Для корректной работы требуется терминал **Kitty**. Если он не установлен, выполните: `brew install --cask kitty`

##  Использование

Базовый запуск отображает системную информацию рядом с изображением:

`apathyfetch`

### Доступные флаги

* **`--hello`** — Активирует режим приветствия. Показывает ваше имя, текущую дату, время, статус батареи и примерное время автономной работы.
  * Пример: `apathyfetch --hello`
* **`--gif`** — Включает анимацию, если в папке с изображениями лежит файл `wall.gif`. Без этого флага отображается только первый кадр.
  * Пример: `apathyfetch --gif`
* **Комбинирование** — Вы можете использовать оба флага одновременно для анимированного приветствия.
  * Пример: `apathyfetch --hello --gif`

##  Требования

* **ОС:** macOS (Big Sur и новее)
* **Терминал:** Kitty (обязательно для вывода графики)
* **Shell:** ZSH (стандартный в macOS)

##  Структура проекта

* `apathyfetch` — основной исполняемый скрипт.
* `install.sh` — скрипт автоматической установки и настройки.
* `images/wall.gif` (или `.jpeg`) — файл изображения по умолчанию.

##  Настройка изображения

По умолчанию скрипт ищет файл `wall.gif` или `wall.jpeg` в папке `~/.local/share/apathyfetch/images/`. Вы можете заменить этот файл на любой свой, сохранив имя и расширение, чтобы изменить фон.
