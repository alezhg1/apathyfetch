# apathyfetch 🍎 [kitty terminal]

Минималистичный и стильный системный фечер для macOS, созданный как современная альтернатива neofetch. Скрипт использует нативный протокол графики терминала **Kitty** для рендеринга изображений (включая анимированные GIF) или универсальный ASCII-режим через `jp2a`.

---

## ✨ Особенности

* **Нативная графика Kitty:** Прямой вывод изображений и GIF-анимации высокого качества (по умолчанию).
* **ASCII Режим:** Ретро-стиль с использованием утилиты `jp2a` (флаг `--ascii`). Работает даже если Kitty не поддерживает графику.
* **Режим анимации:** Поддержка живых обоев через флаг `--gif`.
* **Приветственный режим:** Персонализированный вывод с датой, временем и умным расчетом времени работы батареи (`--hello`).
* **Deep Clear:** Полная очистка экрана и буфера прокрутки перед каждым запуском.
* **Умное выравнивание:** Автоматический расчет отступов, чтобы текст никогда не наезжал на изображение.
* **Mac-specific:** Глубокая интеграция с macOS (модель MacBook, статус GPU, здоровье батареи).

## ⚡ Быстрая установка

Самый простой способ — выполнить одну команду в терминале. Скрипт автоматически скачает файлы, настроит алиас и проверит зависимости.

```bash
curl -fsSL https://raw.githubusercontent.com/alezhg1/apathyfetch/master/install.sh   bash
source ~/.zshrc


Важно: Для корректной работы графики требуется терминал Kitty. Если он не установлен:
bash
Copy

brew install --cask kitty



Использование
Базовый запуск отображает системную информацию рядом с изображением:
bash
Copy

apathyfetch

Доступные флаги


  
    
      Флаг
      Описание
      Пример
    
  
  
    
      --hello
      Режим приветствия с именем, датой, временем и батареей.
      apathyfetch --hello
    
    
      --gif
      Включает анимацию, если в папке images/ есть файл wall.gif. Без флага показывается только статика.
      apathyfetch --gif
    
    
      --ascii
      Выводит изображение в виде цветного ASCII-арта (требует jp2a). Игнорирует нативную графику Kitty.
      apathyfetch --ascii
    
  


Комбинирование флагов
Вы можете использовать несколько флагов одновременно:
bash
Copy

# Анимированное приветствие в нативном режиме
apathyfetch --hello --gif

# Приветствие в стиле ретро (ASCII)
apathyfetch --hello --ascii

# ASCII-арт из первого кадра гифки
apathyfetch --gif --ascii


Для режима --ascii: Требуется установка утилиты jp2a:
bash
Copy

brew install jp2a



 Требования

ОС: macOS (Big Sur и новее)
Терминал: Kitty (рекомендуется для нативной графики)
Shell: ZSH (стандартный в macOS)
Опционально: jp2a (для режима --ascii)

Структура проекта

apathyfetch — основной исполняемый скрипт.
install.sh — скрипт автоматической установки и настройки.
images/ — папка с изображениями:

wall.jpeg (или .jpg, .png) — статичное изображение по умолчанию.
wall.gif — анимированное изображение (активируется флагом --gif).


Настройка изображения
По умолчанию скрипт ищет файлы в папке ~/.local/share/apathyfetch/images/.

Статичный фон: Замените файл wall.jpeg (или создайте wall.png, wall.jpg) на свое изображение.
Анимация: Добавьте файл wall.gif в эту же папку.

При запуске без флагов используется статичный файл (wall.jpeg).
При запуске с флагом --gif используется анимация (wall.gif).
