# apathyfetch 🍎

Минималистичный и стильный системный фечер для **macOS**, оптимизированный под терминал **Kitty**. 
В отличие от стандартного neofetch, этот скрипт использует нативные возможности Kitty для рендеринга изображений и динамически подстраивает текстовые блоки.

## Особенности
- 🖼 **Kitty Graphics Protocol**: Прямой вывод изображений без ASCII-арта.
- 🧹 **Deep Clear**: Полная очистка экрана и буфера прокрутки перед выводом.
- 📏 **Auto-alignment**: Автоматический расчет отступов текста в зависимости от размера картинки.
- 🔋 **Mac-specific**: Вывод модели MacBook, здоровья батареи и параметров GPU через системные утилиты Apple.

## Быстрая установка

Склонируйте репозиторий и запустите инсталлятор:

```bash
git clone https://github.com
cd apathyfetch
chmod +x install.sh
./install.sh
Используйте код с осторожностью.

После установки перезапустите терминал или выполните source ~/.zshrc.
Использование
Просто введите в терминале:
bash
apathyfetch
Используйте код с осторожностью.

Требования
OS: macOS (протестировано на Big Sur и новее)
Terminal: Kitty
Shell: ZSH (дефолтный в macOS)
Структура проекта
apathyfetch — основной исполняемый скрипт.
install.sh — скрипт автоматической настройки алиасов.
